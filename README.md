## Описание работы
В данной лабораторной работе реализованы три основные задачи, направленные на освоение модульного тестирования, работу с алгоритмами поиска/вставки в B+ деревьях и построение доменных моделей.

---

## Задание 1: Математическая функция
Реализован модуль для вычисления значения функции `arctg(x)` через разложение в степенной ряд Маклорена.
- **Особенности:** Для проверки точности используются табличные значения, полученные без использования стандартной библиотеки `java.lang.Math`.
- **Тестирование:** Использованы параметризованные тесты JUnit 5 для проверки различных диапазонов входных данных.
- **Покрытие:** 100% строк и веток.

## Задание 2: Алгоритм B+ Tree
Реализован программный модуль для работы с B+ деревьями с максимальным количеством элементов в ключе $m = 7$.
- **Функционал:** Вставка элементов (`insert`) и поиск (`search`).
- **Специфика:** Реализована логика разделения (split) как листовых, так и внутренних узлов при переполнении.
- **Тестирование:** Тесты покрывают все характерные точки алгоритма, включая краевые случаи при спуске по дереву и переходы по связям между листьями.
- **Покрытие:** Достигнуто **100% Branch Coverage**.

## Задание 3: Доменная модель
На основе заданного текста (фрагмент про Форда, Артура и желтую рыбку) сформирована доменная модель.
- **Сущности:** `Person` (Артур, Форд), `Item` (Флакончик, Рыбка, Хлопья), `Environment`.
- **UML:** В проект включена диаграмма состояний, описывающая переходы Артура между состояниями растерянности и уверенности.
- **Тестирование:** Бизнес-логика модели полностью покрыта тестами. Модель работает автономно и не выводит текст в консоль (согласно требованиям).
